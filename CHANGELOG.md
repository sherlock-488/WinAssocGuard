# Changelog

All notable changes to this project are documented in this file.

The format is based on Keep a Changelog and follows Semantic Versioning.

## [Unreleased]

### Added
- Settings page: monitor interval, notifications toggle, guard toggle, startup toggle.
- Logs page: recent events with extension filtering.
- App picker baseline editor (human-readable app list with optional advanced ProgId input).
- GitHub release workflow (build Windows executable from tag).

### Changed
- Tray menu simplified to two items: open control panel, switch language.
- Main table now shows extension + baseline only (status column removed).

## [0.1.0] - 2026-02-24

### Added
- First public version of WinAssocGuard v4.
- Windows tray app for monitoring file association drifts and auto-restoring baselines.
- Chinese/English UI, extension management, import presets, baseline editing.
